## Descripción
Este proyecto consiste en la implementación de un sistema de gestión de biblioteca universitaria utilizando Python y Programación Orientada a Objetos (POO).

## Estructura del Caso de Estudio
El desarrollo del sistema se realizó siguiendo cuatro fases principales:

1. Requerimientos: Identificación de las necesidades del sistema a partir de una narrativa.
2. Análisis: Aplicación de abstracción para definir entidades, atributos y comportamientos.
3. Diseño: Elaboración del diagrama de clases UML con relaciones y cardinalidades.
4. Desarrollo: Implementación del sistema en Python, aplicando encapsulamiento, herencia y polimorfismo.
